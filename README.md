This is a template for a Nextjs application with Supabase Authentication setup.
## Getting Started

```
git clone https://github.com/wizzzytm/next-supabase-auth
cd next-supabase-auth
pnpm i
```
## Environment Variables
To run the app, ensure the following environment variables are set:
```
DATABASE_URL=""
NEXT_PUBLIC_SUPABASE_URL=""
NEXT_PUBLIC_SUPABASE_ANON_KEY=""
```
